# stardew-checker

Stardew Checker is a way to automate checking if the Stardew Valley board game is in stock. Checked with Cypress, run with GitHub Actions, and notifications through Twilio.

## Installation

Install the dependencies.

`npm install`

Run the script.

`npm test`
